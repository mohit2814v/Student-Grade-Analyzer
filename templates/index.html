System Overview

EduPredict ML is a local, offline classroom analytics platform built for educators, schools, and academic advisors. The system predicts final grades, evaluates student pass-rates, and details study habit corrections using 16 advanced machine learning algorithms.

By operating entirely offline on SQLite and local file systems, the tool enforces strict data privacy, ensuring no student profiles are ever leaked to external cloud platforms or unauthorized APIs.
